一、三种工作模式

  • 命令模式 –默认模式
  • 编辑模式
  • 末行模式

    命令模式进入编辑模式: AIOSaios

    编辑模式进入命令模式: esc

    末行模式进入命令模式: esc*2

    命令模式进入末行模式: :(冒号)

    1.命令模式

    光标移动

  • 左 h
  • 下 j
  • 上 k
  • 右 l
  • 行首 0
  • 行尾 $
  • 文件头 gg
  • 文件尾 G
  • 跳转到300行 300G 300gg

    删除操作

  • 删除字符 x
  • 删除前一个字符 X
  • 删除单词 dw
  • 删除光标到行首 d0
  • 删除光标到行尾 D(d$)
  • 删除当前行 dd
  • 删除4行 4dd

    vim中删除的本质是剪切

    撤销操作

  • 撤销 u
  • 反撤销 ctrl+r

    复制

  • 复制一行 yy
  • 复制4行 4yy

    粘贴

  • 粘贴到光标下一行 p
  • 粘贴到光标上一行 P

    可视模式 v

  • 选择内容 hjkl
  • 复制 y
  • 删除/剪切 d

    可视模式不会复制回车换行，粘贴时直接粘贴到光标位置

    查找

  • 向后查找 /hello
  • 向前查找 ?hello
  • 下一个 n
  • 上一个 N
  • 标记高亮 #

    字符替换

  • 单字符替换 r

    缩进

  • 向右缩进 >>
  • 向左缩进 <<

    man文档

  • 查找慢文档shift+k

    2.文本模式

  • a 光标后插入
  • i 光标前插入
  • A 行尾插入
  • I 行首插入
  • o 下一行插入
  • O 上一行插入
  • s 删除光标字符插入
  • S 删除一行插入

    3.末行模式

  • 跳转到第300行 :300
  • 替换指定行 :20,25s/tom/jerry/g
  • 替换全文 %s/tom/jerry/g
  • 执行命令 :!pwd
  • 强制保存退出 :wq! 快捷键ZZ

    二、分屏

  • 水平分屏 :sp filename
  • 垂直分屏 :vsp filename
  • 退出所有 ：qall
  • 切换 ctrl+w*2

    三、vim配置文件

  • 系统配置文件/etc/vim/vimrc
  • 用户配置文件~/.vim/vimrc
  • set nu 行号
  • set ts=4 tab 设置为4个字符
  • set expandtab 设置按tab默认为softtab
